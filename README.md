# 行业洞察 - 各账号套餐内容追踪

一个用于追踪多个公众号（嬉游、旅行雷达、精明常旅客、Aires、ikids）旅游套餐内容的数据展示面板。

## 在线访问

https://你的用户名.github.io/content-dashboard/

## 功能特点

- 数据展示：展示日期、公众号、分类、目的地、景点、酒店、平台、链接等信息
- 筛选功能：按公众号、平台筛选
- 搜索功能：支持目的地、景点、酒店搜索
- 行间距调整：紧凑/标准/宽松三种模式
- 响应式设计：适配桌面和移动设备

## 管理后台

访问 `content-admin.html` 可以添加、编辑、删除内容数据。

## 技术栈

- HTML5 + CSS3
- Tailwind CSS (CDN)
- Vanilla JavaScript
- GitHub Pages 托管
